# Data-Cleaning-Reporting-Automation

## Project Overview
This project demonstrates data cleaning and reporting automation using Power BI and Excel.

## Features
- Data cleaning using Power Query
- Removed duplicates
- Handled missing values
- Automated reporting
- KPI Cards
- Interactive charts
- Department slicer

## Tools Used
- Power BI
- Microsoft Excel

## Dashboard
The dashboard includes:
- Total Employees
- Average Salary
- Maximum Salary
- Total Salary
- Employees by Department
- Average Salary by Department
- Department Distribution
